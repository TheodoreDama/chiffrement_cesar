Chiffrement de César

Pour crypter ou décrypter, le texte saisi subira les règles suivantes pour chaque caractère :
  • S'il s'agit d'une lettre, elle sera remplacer par une lettre de la même casse (majuscule ou minuscule).
  • S'il s'agit d'un espace, il sera conservez.
  • S'il s'agit d'autre chose( caractère, chiffre, accent), il n'apparaît dans la sortie.
  • Le code doit être un entier compris entre 1 et 25
Par exemple, si nous avions les alphabets ci-dessous (le code est 16) :
l'alphabet en clair : ABCDEFGHIJKLMNOPQRSTUVWXYZ
alphabet chiffré : KLMNOPQRSTUVWXYZABCDEFGHIJ
Et ils ont entré "Writing code is,!;~;;?5555 Cool", nous obtenons "Gbsdsxq myno sc Myyv" comme résultat, car nous préservons les espaces, mais en supprimant tout autre caractère non trouvé dans l'alphabet.
